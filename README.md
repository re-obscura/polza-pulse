# Polza Pulse

Монитор расхода API [Polza.ai](https://polza.ai) в системном трее Windows.

Приложение подключается к API Polza.ai по ключу, периодически опрашивает историю генераций и показывает агрегированную статистику: траты за день / неделю / месяц, дневной лимит, прогноз расхода, топ моделей. Живёт в трее, не требует открытого окна браузера.

---

## Функциональность

### Главное окно

Открывается кликом по иконке в трее. Содержит:

- **Расход за сегодня** — сумма в рублях за текущий день, остаток до лимита
- **Прогресс-бар дневного лимита** — цветовая индикация: зелёный (норма), жёлтый (от 75%), красный (100%+)
- **Метрики расхода** — за 1, 7 и 30 дней
- **Спарклайн** — график дневного расхода за 30 дней; при наведении — точное значение и дата
- **Топ-3 модели** — расход на модели за последние 7 дней
- **Прогноз** — оценка «на сколько дней хватит баланса» на основе медианного или среднего дневного расхода

### История генераций

Отдельное окно: ПКМ по иконке в трее → «История и экспорт». Таблица всех генераций с фильтрацией:

- Фильтры по типу (chat, image, video, audio), модели, статусу и диапазону дат
- Сортировка по дате и расходу
- Детализация каждой генерации: токены, задержка, причина завершения
- **Экспорт в CSV**

### Разбивка по моделям

Отдельное окно: ПКМ по иконке → «Разбивка по моделям». Статистика по моделям:

- Список всех использованных моделей с суммой расхода и количеством генераций
- Период: 7 или 30 дней
- Сводка: число моделей, общий расход, количество генераций

### Настройки

- **API-ключ** — ввод и валидация ключа Polza.ai (`pza_...`)
- **Дневной лимит** — порог расхода за день (по умолчанию 4000 руб.)
- **Интервал опроса** — 1 мин / 5 мин / 15 мин / 1 час
- **Режим прогноза** — среднее или медиана (медиана игнорирует выбросы)
- **Тема** — светлая, тёмная или системная (следует настройкам ОС)
- **Base URL** — адрес API (по умолчанию `https://polza.ai/api/v1`)

### Автообновление

При запуске приложение проверяет GitHub Releases на наличие новой версии. Если обновление доступно — скачивает в фоне. Установка происходит при выходе из приложения (ПКМ по иконке в трее → «Выход»).

### Хранение данных

- API-ключ и настройки сохраняются локально через `electron-store`
- Кэш последнего опроса позволяет видеть данные без доступа к сети
- Никакие данные не передаются третьим сторонам

## Установка

1. Скачать `Polza Pulse Setup x.x.x.exe` со страницы [Releases](https://github.com/re-obscura/polza-pulse/releases/latest)
2. Запустить установщик
3. После установки иконка появится в системном трее

### Первый запуск

1. Клик по иконке в трее → главное окно
2. Кнопка «Настройки» (⚙)
3. Ввести API-ключ Polza.ai, нажать «Сохранить»
4. Приложение начнёт опрос API с интервалом 5 минут

API-ключ можно получить в личном кабинете [Polza.ai](https://polza.ai): Настройки → API-ключи → Создать ключ.

## Системные требования

- Windows 10 или новее, x64
- Около 100 МБ на диске
- Интернет-соединение

---

## Удаление

Стандартное: Параметры Windows → Приложения → Polza Pulse → Удалить.

Данные приложения (ключ, настройки, кэш) хранятся в `%APPDATA%/polza-monitor`. Папку можно удалить вручную для полной очистки.
