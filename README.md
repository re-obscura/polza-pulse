# Polza Pulse

**Монитор расхода и лимитов [Polza.ai](https://polza.ai) в системном трее Windows.**

Polza Pulse живёт в трее и в реальном времени показывает, сколько вы тратите на генерации через API Polza.ai. Баланс, дневной лимит, прогноз «на сколько дней хватит», топ моделей по расходу, история всех генераций и экспорт в CSV — всё в одном компактном окне, которое не мешает работать.

## Возможности

### 💰 Главный экран

Окно открывается по клику на иконку в трее. На одном экране собрано всё самое важное:

- **Сегодня потрачено** — крупная сумма расхода за текущий день и остаток до дневного лимита
- **Прогресс-бар** — наглядная шкала использования лимита: зелёный → жёлтый (75%) → красный (100%+)
- **Расход за 1 / 7 / 30 дней** — три метрики, чтобы понимать динамику трат
- **Спарклайн** — график расхода по дням за последние 30 дней с тултипом при наведении
- **Топ-3 модели** — на какие модели уходит больше всего денег за неделю
- **Прогноз** — «баланса хватит на N дней» на основе вашего среднего или медианного дневного расхода

### 📋 История генераций

Отдельное окно (ПКМ по иконке → «История и экспорт»). Таблица со всеми генерациями:

- Фильтрация по типу (чат, изображение, видео, аудио), модели и статусу
- Сортировка по дате и расходу
- Детальный просмотр каждой генерации (токены, задержка, причина завершения)
- **Экспорт в CSV** — все данные в один клик для анализа в Excel или Google Sheets

### 📊 Разбивка по моделям

Отдельное окно (ПКМ по иконке → «Разбивка по моделям»). Детальная статистика:

- Список всех использованных моделей с суммой расхода и количеством генераций
- Выбор периода: 7 или 30 дней
- Общая сводка: количество моделей, общий расход, число генераций

### 🎨 Персонализация

- **Три темы** — светлая, тёмная и системная (следует настройкам Windows)
- **Настраиваемый интервал опроса** — 1 минута, 5 минут, 15 минут или 1 час
- **Дневной лимит** — установите порог, и приложение предупредит когда расход к нему приблизится
- **Режим расчёта прогноза** — среднее или медиана (медиана устойчивее к выбросам — одна дорогая видео-генерация не исказит картину)
- **Информация в тултипе трея** — при наведении на иконку видно версию, статус лимита и текущие траты

### 🔄 Автообновление

Приложение само проверяет наличие новых версий при запуске. Когда выходит обновление — скачивает его в фоне и устанавливает при следующем выходе из приложения. Вам не нужно ничего делать руками.

### 🛡️ Безопасность и надёжность

- **API-ключ хранится только на вашем устройстве** — в локальном зашифрованном хранилище Electron
- **Никакие данные не отправляются** никуда, кроме официального API Polza.ai
- **Офлайн-устойчивость** — если нет интернета, приложение показывает последние закэшированные данные
- **Один экземпляр** — повторный запуск приложения просто откроет уже работающее окно

## Установка

1. Скачайте последний установщик со страницы [Releases](https://github.com/re-obscura/polza-pulse/releases/latest)
2. Запустите `Polza Pulse Setup x.x.x.exe`
3. После установки приложение появится в системном трее (рядом с часами)

### Первый запуск

1. Кликните на иконку в трее — откроется главное окно
2. Нажмите **⚙** (настройки)
3. Вставьте ваш API-ключ Polza.ai (начинается с `pza_...`)
4. Нажмите «Сохранить» — приложение проверит ключ и начнёт опрашивать API

Готово! Данные обновляются автоматически каждые 5 минут. Интервал можно изменить в настройках.

### Где взять API-ключ?

API-ключ можно получить в личном кабинете [Polza.ai](https://polza.ai). Настройки → API-ключи → Создать ключ.

## Системные требования

- **Windows 10** или новее (64-бит)
- **~100 МБ** свободного места на диске
- Интернет-соединение для работы с API Polza.ai

## Часто задаваемые вопросы

### Приложение не показывает данные

Проверьте API-ключ в настройках. Если ключ введён правильно, но данные не приходят — возможно, на балансе нет средств или API временно недоступен. Нажмите «Обновить» для принудительного опроса.

### Прогноз показывает «недостаточно данных»

Прогнозу нужна история генераций хотя бы за несколько дней. Если вы только начали пользоваться Polza.ai или ключ новый — подождите, данные накопятся.

### Как посмотреть расход за конкретный день?

Откройте «История и экспорт» (ПКМ по иконке в трее), выберите нужные даты в фильтрах.

### Приложение не обновляется автоматически

Автообновление проверяется при каждом запуске. Также можно нажать «Проверить» в настройках. Если обновление найдено — оно применится при выходе из приложения (ПКМ по иконке → Выход).

### Как полностью удалить приложение?

Как обычную программу: Параметры Windows → Приложения → Polza Pulse → Удалить. Данные (ключ, настройки, кэш) хранятся в `%APPDATA%/polza-monitor` — эту папку можно удалить вручную, если хотите стереть всё.
