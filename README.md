# Polza Pulse

Монитор расхода и лимитов [Polza.ai](https://polza.ai) в системном трее Windows. Видит баланс, траты по дням, прогноз «на сколько дней хватит» и топ моделей — всё в одном компактном окне.

![Polza Pulse](icons/icon-128.png)

## Возможности

- **Баланс и расход** — текущий баланс, траты за сегодня / 7 / 30 дней
- **Дневной лимит** — настраиваемый порог с прогресс-баром и предупреждениями
- **Прогноз** — «баланса хватит на N дней» (по медиане или среднему расходу)
- **Спарклайн** — график расхода по дням за 30 дней
- **Топ моделей** — разбивка трат по моделям (GPT-4o, Claude и т.д.)
- **История и экспорт** — полная история генераций в отдельном окне
- **Светлая / тёмная / системная тема**
- **Автообновление** — само качает новые версии из GitHub Releases
- **Офлайн-устойчивость** — кэширует последние данные, показывает даже без сети

## Установка

Скачай последний `.exe`-установщик со страницы [Releases](https://github.com/re-obscura/polza-pulse/releases/latest) и запусти.

После установки:
1. Открой приложение (иконка появится в трее)
2. Зайди в Настройки (⚙)
3. Вставь свой API-ключ Polza.ai (`pza_...`)
4. Приложение начнёт опрашивать API каждые 5 минут

## Разработка

```bash
# Установка зависимостей
npm ci

# Сборка и запуск в dev-режиме
npm run dev

# Watch-режим (пересборка при изменениях)
npm run watch

# Проверка типов
npm run typecheck

# Сборка установщика (electron-builder)
npm run dist
```

### Структура проекта

```
src/
  main/          # Electron main process (точка входа, трей, IPC, опрос API, автообновление)
    index.ts     # Точка входа: окно, трей, запуск polling и проверка обновлений
    tray.ts      # Системный трей: иконка, tooltip, контекстное меню
    poller.ts    # Периодический опрос Polza.ai API
    store.ts     # Персистентное хранилище через electron-store
    updater.ts   # Автообновление через electron-updater (GitHub Releases)
    ipc.ts       # IPC-обработчики между main и renderer
  preload/
    index.ts     # contextBridge: безопасный мост между main и renderer
  renderer/
    popup/       # Главное окно (баланс, лимит, спарклайн, топ моделей)
    history/     # Окно истории генераций и экспорта
    models/      # Окно разбивки по моделям
    styles/      # Общие стили (токены, база, компоненты)
    lib/         # Бизнес-логика (клиент API, агрегация, прогноз, форматирование)
    ui/          # UI-компоненты (спарклайн, тема)
  types.ts       # Общие типы и интерфейсы
scripts/
  build.mjs      # Сборка через esbuild (main + preload + renderer)
  make-icons.mjs # Генерация иконок
icons/           # Иконки приложения (16–128px)
```

### Технологии

| Слой | Технология |
|------|-----------|
| Рантайм | Electron 31 |
| Язык | TypeScript |
| Сборка | esbuild |
| Упаковка | electron-builder (NSIS-установщик) |
| Автообновление | electron-updater → GitHub Releases |
| Хранилище | electron-store |
| UI | Vanilla HTML/CSS/TS (без фреймворков) |

## Релиз и автообновление

1. Подними версию в `package.json`
2. Создай тег и запушь:
   ```bash
   git tag v1.0.1
   git push --tags
   ```
3. GitHub Actions соберёт `.exe` и опубликует релиз
4. Пользователи получат обновление автоматически при следующем запуске
